\subsection{\EN{\main function sole argument is array too}\RU{Единственный аргумент \main это также массив}}

\EN{Let's use the sole \main function argument, which is array of strings:}
\RU{Будем использовать единственный аргумент \main, который массив строк:}

\begin{lstlisting}
public class UseArgument
{
	public static void main(String[] args)
	{
		System.out.print("Hi, ");
		System.out.print(args[1]);
		System.out.println(". How are you?");
	}
}
\end{lstlisting}

\EN{Zeroth argument is a program name (like in \CCpp, etc), 
so the 1st argument supplied by user is 1st.}
\RU{Нулевой аргумент это имя программы (как в \CCpp, итд),
так что первый аргумент это тот, что пользователь добавил первым.}

\begin{lstlisting}

\EN{\TT{aload\_0} at 11 loads \IT{reference} of the zeroth \ac{LVA} slot 
(1st and only \main argument).}
\RU{\TT{aload\_0} на 11 загружают \IT{reference} на нулевой слот \ac{LVA} 
(первый и единственный аргумент \main).}
\EN{\TT{iconst\_1} and \TT{aaload} at 12 and 13 takes \IT{reference} to the first (counting at 0) 
element of array.}
\RU{\TT{iconst\_1} и \TT{aaload} на 12 и 13 берут \IT{reference} на первый (считая с 0) 
элемент массива.}
\EN{The \IT{reference} to string object is at \ac{TOS} at offset 14, and it is 
taken from there by \TT{println} method.}
\RU{\IT{Reference} на строковый объект на \ac{TOS} по смещению 14, и оттуда он 
берется методом \TT{println}.}
