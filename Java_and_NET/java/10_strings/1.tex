\subsection{\EN{First example}\RU{Первый пример}}

\EN{Strings are objects and constructed in the same way as other objects (and arrays).}
\RU{Строки это объекты, и конструируются так же как и другие объекты (и массивы).}

\begin{lstlisting}
	public static void main(String[] args)
	{
		System.out.println("What is your name?");
		String input = System.console().readLine();
		System.out.println("Hello, "+input);
	}
\end{lstlisting}

\EN{\TT{readLine()} method is called at offset 11, \IT{reference} to string (which supplied by user) 
is then stored at \ac{TOS}.}
\RU{Метод \TT{readLine()} вызывается по смещению 11, \IT{reference} на строку (введенную пользователем) 
остается на \ac{TOS}.}
\EN{At offset 14 \IT{reference} to string is stored to slot 1 of \ac{LVA}.}
\RU{По смещению 14, \IT{reference} на строку сохраняется в первом слоте \ac{LVA}.}
\EN{The string user entered is reloaded at offset 30 and concatenated with the \q{Hello, } string
using \TT{StringBuilder} class.}
\RU{Строка введенная пользователем перезагружается по смещению 30 и складывается со строкой \q{Hello, }
используя класс \TT{StringBuilder}.}
\EN{Constructed string is then printed using \TT{println} method at offset 37.}
\RU{Сконструированная строка затем выводится используя метод \TT{println} по смещению 37.}
