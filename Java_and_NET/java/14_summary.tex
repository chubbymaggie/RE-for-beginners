\section{\EN{Summary}\RU{Итоги}}

\EN{What is missing in Java in comparison to \CCpp?}
\RU{Чего не хватает в Java в сравнении с \CCpp?}

\begin{itemize}
\item \EN{Structures: use classes}\RU{Структуры: используйте классы}.

\item \EN{Unions: use class hierarchies}\RU{Ообъединения (union): используйте иерархии классов}.

\item \EN{Unsigned data types}\RU{Беззнаковые типы данных}.
\EN{By the way, this makes cryptographical algorithms somewhat harder to implement in Java.}
\RU{Кстати, из-за этого реализовывать криптографические алгоритмы на Java немного труднее.}

\item \EN{Function pointers}\RU{Указатели на функции}.
\end{itemize}
